# Erebos  

## Concepts
Erebos is a platformer in 2.5D view.  
With a wide range of actions, the character is able to find his way into the levels.  
  
The context is about Greek mythology, where Athena is banned in the Erebos.  
  
  
## Team & Constraints
- one month 
- 3 game designers
- 4 game artists
